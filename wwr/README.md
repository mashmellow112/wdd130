# White Water Rafting Co. - Adventure on the Nile

A modern, responsive multi-page website built for the **WDD 130** Web Development course. This project showcases professional UI/UX design, clean CSS architecture, and a focused user experience for an adventure tourism company based in **Jinja, Uganda**.

## 🛠️ Built With
* **HTML5** - Semantic structure for accessibility and SEO.
* **CSS3** - Custom properties (variables), Flexbox, and CSS Grid for a modern layout.
* **Cinematic UI** - Focus on high-contrast visuals, dark mode elements, and glassmorphism.

## 📁 Project Structure
* `index.html`: Home page featuring a cinematic hero section and call-to-action.
* `about.html`: Company history, mission statement, and team values.
* `trips.html`: Detailed catalog of rafting packages on the River Nile (Grade 5 rapids).
* `contact.html`: Interactive contact form with modern styling and location details.
* `styles/rafting.css`: Centralized stylesheet for site-wide consistency.

## ✨ Key Features
* **Responsive Design:** Optimized for mobile, tablet, and desktop views using media queries.
* **Visual Hierarchy:** Strategic use of typography and whitespace to guide user attention.
* **Local Context:** Content specifically tailored to the Ugandan white water rafting industry.
* **Modern Branding:** Custom logo integration and a "Dark & Gold" color palette for a premium feel.
